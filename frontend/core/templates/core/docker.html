This is a tutorial to run predictions from our official Docker volume.
There are no restrictions on the number of sequences that can be run

Parameters

(Mandatory) input_file: Local path to te input file

(Optional)'--output_file', '-o': Name of the output file

(Optional)'--step_size', '-s': Value of the desired step size (used for sequences with length >60)

Pull the Docker image:

docker pull sganzg/clade_predictor:latest

Build the Docker image

docker build -t local_prediction_tool .

Run with results on screen (basic)

docker run --rm -v /path/to/your/fasta:data local_prediction_tool /data/your_fasta.fasta

Run saving results as a csv in the same directory of your input file

docker run --rm -v /path/to/your/fasta:data local_prediction_tool /data/your_fasta.fasta -o /data/results.csv

Run inputting the desired value of step size(for sequences with length > 60)

docker run --rm -v /path/to/your/fasta:data local_prediction_tool /data/your_fasta.fasta -o /data/results.csv -s 4
