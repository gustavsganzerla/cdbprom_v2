Resources

CDBProm v2.0 is fully open-access under a MIT license. If considering using CDBProm v2.0 in our research work, we kindly ask you to cite us. Interested users can further explore CDBProm v2.0 by using our API endpoints to integrate both the predictive framework and the database querying with other tools. In addition, we enable the predictive framework through a Dockerized application to run locally. The fine-tuned DNABERT6 model can be directly downloaded from Hugging Face. All the data enabled for querying within CDBProm v2.0 can be downloaded. We also provide how-to-use tutorials in video format.
